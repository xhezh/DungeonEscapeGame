# Краткое описание проекта:
## Название проекта: “Беглец из подземелья”
## Цель проекта: 
- создать 2D игру на выживание, которая совмещает элементы платформера и башенной защиты, где игрок должен защитить ключевые точки на арене от волн врагов и дойти до финиша, используя собранные ресурсы для строительства башен.
## Краткое описание задач:
- Разработка механики перемещения игрока с помощью клавиш A, D, Space.
- Реализация системы строительства 3 типов башен с разными эффектами (урон, замедление, взрыв).
- Программирование противников с разными типами поведения (маленький и быстрый враг, обычный, большой и медленный).
- Создание уровней с платформами, ресурсами и противниками.
- Внедрение механики сбора ресурсов (золота) и использования их для постройки башен.
- Реализация системы волн врагов с нарастающей сложностью.
- Настройка логики окончания уровня (финиш, победа/поражение, перезапуск).
- Разработка пользовательского интерфейса (UI): отображение здоровья, золота, волны, меню паузы и окончания игры.
- Добавление звуков и музыки, регулировка громкости через меню настроек.
- Сохранение прогресса игрока: пройденные уровни, настройки громкости.
- Реализация системы меню: главное меню, меню уровней, настройки.
- Поддержка системы перехода между сценами и загрузки следующего уровня.
- Интеграция анимаций (бег, прыжок, атака, получение урона).
- И еще многие более простые механики.
